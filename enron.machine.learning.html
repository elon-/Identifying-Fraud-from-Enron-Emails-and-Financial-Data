def fraction (total_message,message_with_poi):
    if total_message!="NaN"and message_with_poi!="NaN":
            fraction=(float(message_with_poi)/float(total_message))
    else:
         fraction=0.        
    return fraction

conclustion:¶

The Decision Tree we have used have a very decent precision and recall. I prefer precision over recall because if our model make some false prediction it is very easy for a person using our model to stop believing and even stop using my model. So I believe to have model with high precision at it initial testing.Thus the Decision Tree will be the opt model for this data set.
